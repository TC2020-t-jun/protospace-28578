# テーブル設計

## users テーブル

| Column     | Type   | Options     |
| ---------- | ------ | ----------- |
| email      | string | null: false |
| password   | string | null: false |
| name       | string | null: false |
| profile    | text   | null: false |
| occupation | text   | null: false |
| position   | text   | null: false |

## comments テーブル

| Column     | Type   | Options     |
| ---------- | ------ | ----------- |
| text       | text   | null: false |
| user       | references |         |
| prototype  | references |         |


## rooms テーブル

| Column | Type   | Options     |
| ------ | ------ | ----------- |

## room_users テーブル

| Column | Type       | Options                        |
